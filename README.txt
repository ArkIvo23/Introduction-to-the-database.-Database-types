Пояснение к схеме:

- Таблица "Жанры" содержит информацию о музыкальных жанрах, имеет поле "ID_Жанра" для уникальной идентификации жанра и поле "Название_Жанра" для отображения названия жанра на сайте.
- Таблица "Исполнители" хранит данные об исполнителях, включая их имена (псевдонимы) в поле "Имя_Исполнителя". Также есть поле "ID_Жанра", которое связывает исполнителя с соответствующим жанром из таблицы "Жанры".
- В таблице "Альбомы" содержится информация о музыкальных альбомах, включая название, год выпуска и поле "ID_Исполнителя", связывающее альбом с исполнителем из таблицы "Исполнители".
- В таблице "Треки" хранится информация о конкретных треках, их названиях, длительности и поле "ID_Альбома", которое связывает трек с альбомом из таблицы "Альбомы".
